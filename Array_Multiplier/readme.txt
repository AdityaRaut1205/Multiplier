# Array Multiplier

## Description
This project implements an **Array Multiplier** using Verilog, including a **Partial Product Generator** and **Accumulation Unit**.  
The design can be simulated using the provided testbench files.

## Folder Structure
- `src/` : Contains Verilog **module files** for the Array Multiplier.  
- `tb/` : Contains **testbench files** for simulation and verification.

## How to Simulate
1. Open Vivado and create a new project.  
2. Add the module files from `src/` and the testbench files from `tb/`.  
3. Run simulation to verify multiplication results.
